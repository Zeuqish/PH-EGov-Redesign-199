Using Text Block The text block automatically puts in a header when a ui-header element is detected. Text is automatically put in the larger container when the split option is used. For putting content in the smaller container, add the words "left" or "right" on the element; for example, <ui-container left> when the split option is left will make the block place the container on the left, but if the split option is on the right, the ui-container will not render.
